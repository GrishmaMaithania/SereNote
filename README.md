SereNote 🎵

SereNote is a music learning and analysis web application that helps users detect song keys, chords, and vocal pitch, while also providing music theory learning and collaboration features.

Features

* 🎶 Key & chord detection from audio and YouTube
* 🎤 Real-time pitch detection
* 🎼 Music theory lessons with quizzes
* 🎧 Audio merging and collaboration
* 👤 User authentication and profile management

Tech Stack

* Frontend: React.js
* Backend: Django (Python)
* Database: Firebase Firestore

Modules & Libraries Used

Frontend

* React.js
* Axios
* Tailwind CSS
* Framer Motion
* React Router
* Lucide React

Backend

* Django
* Django REST Framework
* NumPy
* Librosa
* PyAudio

Other Tools & Services

* Firebase Authentication
* Firebase Firestore
* WebAssembly (WASM) for audio processing
* GitHub for version control

How to Run

1. Clone the repository
2. Install dependencies
3. Run backend server
4. Run frontend server

---

SereNote aims to bridge the gap between music theory and practical learning through an interactive and beginner-friendly platform.
